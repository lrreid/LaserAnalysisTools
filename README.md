# LaserAnalysisTools
Python package with tools to extract data from diagnostics for laser systems and preform analysis on that data.


## Installation
1) Clone this repository to a directory in your local computer via `git`
```bash
git clone https://github.com/lrreid/LaserAnalysisTools.git
```
alternativly, download the code from [here](https://github.com/lrreid/LaserAnalysisTools/archive/master.zip) and unzip it.

2) In a terminal window, navigate to the unzipped directory and perform the installation with
```bash
pip install ./
```
